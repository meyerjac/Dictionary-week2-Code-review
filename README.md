# _my Dictionary_

#### _add words and definitions to your dictionary, Sept. 16th, 2016_

#### By _**Jackson Meyer**_

## Description

_The purpose of this dictionary is to prove that we are capable of using routing methods, writing tests and writing java in Spark.  Its purpose is to show how to use forms on a very simple level, gather and draw data from them_

## Setup/Installation Requirements

## SPECS

_fill out a form and submit 1 new word_
_have that word populate back in my index as a clickable link_
_be able to repeat this for multiple words_
_when words are clicked on show the current definitions for that word_
_be able to add a definition to a word through a simple form on another page_
_be able to add definitions to all our words_

* _in order to run this app, please clone from github, then open the file and gradle run it, then navigate to an internet browser and load localhost: 4567_
